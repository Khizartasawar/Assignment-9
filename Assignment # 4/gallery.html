HTML5 : Figure And Figure Caption

<figure> </figure>


Images often come with captions. HTML_S has introduced a new element to contain images and their caption so that the two are associated. You can have more than one Image Inside the element as long as they all share the same caption.

<figcaption> </figcaption>


The element has been added to HTML-S in order to allow web page authors to add a caption to an image. Before these elements were created there was no way to associate an element with its caption. Older brou,-sers that do not understand HTML_S elements simply ignore the new elements and display the content of them.
